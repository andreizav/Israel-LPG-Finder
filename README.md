# Israel LPG Finder

**Israel LPG Finder** is a comprehensive mobile application designed to help drivers locate LPG (Autogas) stations across Israel. Built with performance and ease of use in mind, it allows users to find the nearest stations, compare real-time prices, and navigate directly via Waze.

## Features

-   **Station Locator**: extensive database of LPG stations throughout Israel.
-   **Real-Time Prices**: View and compare up-to-date fuel prices.
-   **Smart Navigation**: One-tap integration with Waze for finding the best route.
-   **Dual Language Support**: Fully localized for both Hebrew (RTL) and English.
-   **Filter & Search**: Easily find stations by brand, distance, or price.

## Mobile Application

This project is built as a cross-platform mobile application using **Angular** and **Capacitor**. It is optimized for Android devices and available for local deployment.

### Tech Stack

-   **Frontend**: Angular (Latest)
-   **Mobile Runtime**: Capacitor 7
-   **Styling**: TailwindCSS
-   **Maps**: Leaflet

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Build for Android**:
    ```bash
    npm run build:mobile
    ```
    This command compiles the web assets, syncs them with the Android platform, and opens Android Studio.

3.  **Run in Browser**:
    ```bash
    ng serve
    ```
